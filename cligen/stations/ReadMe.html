The files under this directory are provided by U. S. Forest Service in
Moscow, Idaho.  We believe them to be the best-quality, most extensive
collection of single-station weather parameter files for the U. S. that
currently exists.

The files may be utilized with the recoded CLIGEN as "input files":
- Go to the state of interest under this directory.
- Look in "xxclimates.html" where xx is the two-letter abbreviation for 
   your state.  Find the index number(s) associated with the station(s) 
   of interest.
- Copy xx.par to your directory which contains the Cligen
   executable.
- Run Cligen using the '-i' option.
   EX: cligen -b1 -y30 -iin124715.par -oLAFptest
       Runs a 30-year simulation for Lafayette, Indiana, using 
       "in124715.par" as the station parameter file, and puts
       the output into "LAFptest".
